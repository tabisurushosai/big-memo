# Privacy Policy

おおきなメモは、家族が書いた大きな文字のメモと今日のやることを表示するための Chrome 拡張です。

## データの扱い

- 入力されたメモと今日のやることは、Chrome の `storage.local` にのみ保存されます。
- 本拡張は API 通信、外部サーバー送信、解析サービス送信を行いません。
- 本拡張はホスト権限を要求せず、閲覧中の Web ページ内容を読み取りません。
- 保存データは、ユーザーがブラウザ側で拡張のデータを削除するまで、このブラウザ内に残ります。

## Permissions

使用する権限は `storage` のみです。メモと今日のやること、初回起動日時、Premium 状態をローカル保存するために使います。
